# DecExp
Montecalro Simulation of Experiment at RCNP, December 2016
(GLUT and OpenGL test program)

Extract the zip file and change directory to 'DecExp-master', then make 'DecExp'
```
$ cd ~/Downloads/DecExp-master
$ make
$ ./bin/DecExp
```
